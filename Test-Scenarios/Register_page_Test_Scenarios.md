# Register Page Test Scenarios

## 1. Positive Test Scenarios
-Verify user can register successfully with a valid email and a valid password.

## 2. Nagative Test Scenarios
-Verify an error message is displayed when the email field is empty.
-Verify an error message is displayed when the password field is empty.
-Verify an error message is displayed when an invalid email format is entered.
-Verify an error message is displayed when the password does not meet the required criteria.

## 3. Functional/Navigation Test Scenarios
-Verify clicking on the password eye icon toggles password visibility.
-Verify clicking on the "Register" button submits the form successfully when valid fields are entered.
-Verify a verification email is sent after successful registration.
-Verify clicking on the "Login" link navigates to the Login page

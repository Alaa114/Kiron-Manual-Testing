# Login Page Test Scenarios

## 1. Positive Test Scenarios
-Verify user can log in with a valid email and a valid password.

## 2. Nagative Test Scenarios
-Verify an error message is displayed when email field is empty.
-Verify an error message is displayed when password field is empty.
-Verify an error message is displayed when the user enters an invalid password or email.

## 3. Functional/Navigation Test Scenarios
-Verify clicking on “Login” button navigates to Home page when valid fields are entered.
-Verify clicking on “Forgot your password?” navigates to reset password page.
-Verify clicking on “Register” navigates to registration page.
-Verify password eye icon toggles password visibility.
